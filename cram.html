CH 1Row Reduction · REF / RREF High
Lay 1.1–1.5 · solve any Ax=b

3 elementary row operations (preserve solution set):

  • Replace: $R_i \to R_i + cR_j$
  • Swap: $R_i \leftrightarrow R_j$
  • Scale: $R_i \to cR_i$, $c\ne 0$

Echelon form (REF): zero rows at bottom · each leading entry is right of the one above · entries below pivots = 0.
RREF: leading 1s · only nonzero in their column.

Recipe · Gauss-Jordan
  1. Get leading 1 in row 1, col 1 (swap/scale).
  2. Zero out below the pivot using replace.
  3. Move to next pivot (down-right). Repeat.
  4. Back-substitute upward to clear above pivots.
Pivot column = column with a pivot. Free var = non-pivot column. Solutions exist ⇔ no row $[0\,0\,\dots\,0\,|\,b]$ with $b\ne 0$.
CH 1Span · Independence · Transformations Med
Lay 1.3 / 1.7 / 1.8 / 1.9

Span: $\text{Span}\{v_1,\dots,v_p\}=\{c_1v_1+\dots+c_pv_p\}$. To test if $b\in\text{Span}$: row-reduce $[v_1\,\dots\,v_p\,|\,b]$ — consistent ⇔ yes.

Linear independence: $\{v_1,\dots,v_p\}$ indep ⇔ $Ax=0$ has only the trivial sol ⇔ every col is a pivot col.

  • More vectors than entries (p > n) → always dependent.
  • Any set with $\vec 0$ → dependent.

Linear Transformation $T:\mathbb R^n\to\mathbb R^m$: $T(u+v)=T(u)+T(v)$, $T(cu)=cT(u)$. Standard matrix: $A=[T(e_1)\,T(e_2)\,\dots\,T(e_n)]$.

One-to-one ⇔ columns of $A$ are linearly indep · ⇔ $Ax=0$ has only trivial sol.
Onto ⇔ columns span $\mathbb R^m$ · ⇔ $A$ has a pivot in every row.
CH 2Matrix Inverse · IMT High
Lay 2.2–2.3 · know all 14 equivs

2×2 quick: $\displaystyle A=\begin{bmatrix}a&b\\c&d\end{bmatrix}\Rightarrow A^{-1}=\frac{1}{ad-bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}$, requires $ad-bc\ne 0$.

General method: row-reduce $[A\,|\,I]\to [I\,|\,A^{-1}]$. If left side ≠ $I$ → $A$ is singular.

Properties: $(AB)^{-1}=B^{-1}A^{-1}$ · $(A^T)^{-1}=(A^{-1})^T$ · $(A^{-1})^{-1}=A$.

Invertible Matrix Theorem — for $A$ ($n\times n$), TFAE: (1) $A$ invertible · (2) $A$ row-equiv to $I_n$ · (3) $n$ pivots · (4) $Ax=0$ only trivial · (5) cols linearly indep · (6) $T(x)=Ax$ one-to-one · (7) $Ax=b$ has unique sol $\forall b$ · (8) cols span $\mathbb R^n$ · (9) $T$ onto · (10) $\exists C$ with $CA=I$ · (11) $\exists D$ with $AD=I$ · (12) $A^T$ invertible · (13) cols of $A$ are a basis of $\mathbb R^n$ · (14) $\det A\ne 0$.
Solve $Ax=b$ via inverse: $x = A^{-1}b$.
CH 3Determinants · Cofactor · Cramer High
Lay 3.1–3.3 · easy marks if crisp

2×2: $\det\!\begin{bmatrix}a&b\\c&d\end{bmatrix}=ad-bc$.
3×3 (Sarrus): diag products – anti-diag products.

Cofactor expansion along row $i$: $\det A = \sum_j (-1)^{i+j} a_{ij}\,M_{ij}$, where $M_{ij}$ = minor (det of submatrix). Pick the row/col with most zeros.

Sign pattern: $\begin{bmatrix}+&-&+\\-&+&-\\+&-&+\end{bmatrix}$

Row-op effects on $\det$: Replace = unchanged · Swap = ×(−1) · Scale row by $k$ = ×$k$.
Properties: $\det(AB)=\det A\det B$ · $\det A^T=\det A$ · $\det A^{-1}=1/\det A$ · Triangular: $\det = $ product of diagonal.

Cramer's Rule ($A$ invertible, $Ax=b$): $x_i=\dfrac{\det A_i(b)}{\det A}$ where $A_i(b)$ = $A$ with column $i$ replaced by $b$.

CH 5Eigenvalues · Eigenvectors High
Lay 5.1–5.2 · core of Markov + diag

Definition: $\lambda$ is an eigenvalue of $A$ if $\exists\,v\ne 0$ with $Av=\lambda v$. $v$ = eigenvector for $\lambda$.

Characteristic equation: $\det(A-\lambda I)=0$ → gives the eigenvalues.

Eigenvector: solve $(A-\lambda I)v = 0$ → null space.

Recipe · Find eigenvalues + eigenvectors
  1. Form $A-\lambda I$, take det, set = 0.
  2. Solve polynomial for $\lambda_1,\lambda_2,\dots$.
  3. For each $\lambda$: row-reduce $(A-\lambda I)$ and read off the null space.
  4. Pick any nonzero vector from each null space.
Easy wins: Triangular $A$ → eigenvalues = diagonal entries. Trace $=\sum\lambda$, $\det = \prod\lambda$ — sanity-check your answer.
CH 5Diagonalization $A=PDP^{-1}$ High
Lay 5.3 · powers + Markov

Theorem (Diag): $A$ ($n\times n$) is diagonalizable ⇔ $A$ has $n$ linearly independent eigenvectors.

Sufficient: $n$ distinct eigenvalues ⇒ diagonalizable.
Iff: for every eigenvalue $\lambda$, $\dim(\text{eigenspace}) = $ algebraic multiplicity.

Recipe · Diagonalize
  1. Find all eigenvalues (char eq).
  2. For each $\lambda$ find a basis of its eigenspace.
  3. If total = $n$ indep vectors → build $P=[v_1\,\dots\,v_n]$, $D=\text{diag}(\lambda_1,\dots,\lambda_n)$ in matching order.
  4. Otherwise: state NOT diagonalizable.
Powers cheap: $A^k = PD^kP^{-1}$, where $D^k = \text{diag}(\lambda_1^k,\dots,\lambda_n^k)$.
CH 5⭐Markov Chains · Steady State High
Class addition · examiner favourite

Stochastic matrix $P$: each column sums to 1 and all entries $\ge 0$.

Steady-state vector $q$: probability vector ($q_i\ge 0$, $\sum q_i = 1$) with $Pq=q$, i.e. eigenvector of $\lambda=1$.

Recipe · Steady state (2 ways)
  1. Direct: solve $(P-I)q=0$ subject to $\sum q_i=1$.
  2. Eigen: find eigenvector for $\lambda=1$, then normalize so entries sum to 1.
If $P$ is regular (some power has all positive entries) → unique steady state, and $P^k x_0 \to q$ from any start.
Trap: always check columns sum to 1 BEFORE proceeding. If rows sum to 1 (row-stochastic), use $P^T$ instead.
LPLinear Programming · Graphical High
Budnick · Lectures 17–18

Standard form (max): maximize $z=c^Tx$ s.t. $Ax\le b$, $x\ge 0$.

Recipe · Graphical (2 vars)
  1. Plot each constraint line (intercepts: set the other = 0).
  2. Shade feasible region (test the origin).
  3. Identify corner points (intersections + axis intercepts on boundary).
  4. Evaluate $z$ at every corner.
  5. Pick max (or min) — that's the optimal.
Special cases: unbounded (objective grows forever in feasible region) · infeasible (region empty) · alternate optima (entire edge max).
LPSimplex Method High
Budnick · Lec 20–21

Convert to standard: add slack $s_i\ge 0$ to each ≤, surplus & artificial to each ≥/=.

Recipe · Maximize simplex (Big-M if needed)
  1. Write initial tableau with $c_j$ row, $z_j$ row, $c_j-z_j$ row.
  2. Entering var: most positive $c_j - z_j$ (for max).
  3. Leaving var: smallest non-negative ratio $b_i / a_{ij}$ (positive $a_{ij}$ only).
  4. Pivot: make pivot = 1, zero out rest of column.
  5. Repeat until all $c_j - z_j \le 0$ → optimal.
For min: use $z_j - c_j$, or convert (min $z$ = − max $-z$).
Trap: if a tie on entering, pick any. If tie on leaving — degeneracy possible. If pivot column has no positive entry → unbounded.
LPDual Problem Med
Budnick · Lec 23–24

Primal (max): max $c^Tx$ s.t. $Ax\le b$, $x\ge 0$. Dual (min): min $b^Ty$ s.t. $A^Ty\ge c$, $y\ge 0$.

  • $\#$ primal vars = $\#$ dual constraints, and vice-versa.
  • If primal feasible bounded, both have same optimal value (Strong duality).
  • Use dual when fewer constraints than vars — easier simplex.
TXTransportation · NWCM / Least Cost / VAM / MODI High
Budnick · Lec 25–26

Step 0 — Balance: if $\sum\text{supply}\ne\sum\text{demand}$, add a dummy row/col with cost 0.

Initial BFS — VAM (best)
  1. For each row & col, compute penalty = (2nd lowest cost − lowest cost).
  2. Pick row/col with largest penalty; in it, allocate the max to the lowest-cost cell.
  3. Cross out the satisfied row/col; recompute penalties; repeat.
Optimality — MODI / u–v
  1. For each occupied cell: $u_i + v_j = c_{ij}$. Set $u_1=0$, solve for the rest.
  2. For each empty cell, compute $d_{ij} = c_{ij} - (u_i + v_j)$.
  3. If all $d_{ij}\ge 0$ → optimal. Else pick most-negative $d_{ij}$ as entering cell.
  4. Draw a closed loop (+/−) starting at entering cell; reallocate min of (−) cells.
  5. Repeat MODI.
Degeneracy (occupied cells < $m+n-1$): add an $\epsilon$ (a 0) to an empty independent cell to restore count. Treat as occupied for MODI but value = 0.
ASAssignment · Hungarian Method High
Budnick · Lec 27–28

Setup: square cost matrix ($n$ workers × $n$ jobs). If not square → add dummy row/col with 0 cost.

Recipe · Hungarian (minimize)
  1. Row reduction: subtract row min from every entry in that row.
  2. Col reduction: subtract col min from every entry in that col.
  3. Cover all zeros with the minimum number of lines.
  4. If # lines = $n$ → optimal: pick zero-assignment. Else go to step 5.
  5. Find smallest uncovered entry $k$. Subtract $k$ from all uncovered. Add $k$ to entries covered twice. Repeat from step 3.
For maximization: first convert to min by subtracting every entry from the largest entry, then run Hungarian.
Topics · 1-page summaries

Ch 1 · Linear Systems & Vectors
REF/RREF · Span · Ax=b

Big idea: a system $Ax=b$ has 0, 1, or ∞ solutions. Row-reduce $[A|b]$ to find which.

Must-do: Be fluent with row ops. Always state pivot vs free vars. Write solution in parametric vector form if infinite.

Watch: consistency check — if RREF has a row $[0\,0\,\dots\,0\,|\,c]$ with $c\ne 0$ → inconsistent, no solution. State this explicitly.
Ch 1.7–1.9 · Independence + LT
When are vectors free?

Big idea: Linear independence ↔ unique combinations. LT ↔ matrix multiplication.

Must-do: Find standard matrix via $A=[T(e_1),\dots]$. Check one-to-one (cols indep) and onto (pivot in every row).

Trap: $p > n$ vectors in $\mathbb R^n$ are always dependent — examiners love this freebie. State it as a one-liner.
Ch 2 · Matrix Algebra + Inverse
2×2 formula · IMT recital

Big idea: matrices represent linear maps. Invertible = bijective on $\mathbb R^n$.

Must-do: Recite IMT mid-answer when asked "is A invertible / does Ax=b have unique sol / are cols indep" — they're the same question.

3 ways to invert
  1. 2×2: $\frac{1}{ad-bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}$.
  2. General: row-reduce $[A|I]$.
  3. Via $\det$: $A^{-1}=\frac{1}{\det A}\,\text{adj}(A)$ (cofactor transpose).
Ch 3 · Determinants
Cofactor · Cramer · row-op rules

Big idea: $\det A$ measures invertibility (= 0 ⇔ singular) and oriented volume.

Must-do: Cofactor expand along whichever row/col has the most zeros. Use row-op effects to simplify big matrices first.

Cramer's Rule = formula answer for $x_i$ when $A$ invertible: $x_i = \det A_i(b)/\det A$.
Ch 5 · Eigenvalues + Diagonalization ⭐
PDP⁻¹ · powers · Markov

Big idea: if $Av=\lambda v$, action of $A$ on $v$ = scaling. Diagonalize → easy powers, easy steady-state.

Must-do: Char equation $\det(A-\lambda I)=0$ → solve for $\lambda$. Null-space of $(A-\lambda I)$ → eigenvectors. Need $n$ indep eigenvectors to diagonalize.

Triangular shortcut: eigenvalues = diagonal entries — write that down immediately if matrix is triangular.
Sanity check: $\text{trace} = \sum\lambda$ and $\det = \prod\lambda$. If they don't match, recompute.
LP · Graphical + Simplex + Dual
Pick the right tool

Big idea: optimum of a linear objective over a polytope sits at a corner. Simplex walks the corners.

Must-do: Always start with standard form (slacks). Read the tableau carefully — entering = most + $c_j-z_j$ for max, leaving = min ratio.

Trap: sign conventions differ in slides — confirm if you're using $c_j-z_j$ (most +ve enters) or $z_j-c_j$. Stay consistent within a question.
Transportation + Assignment
VAM → MODI → done · Hungarian

Big idea: ship $x_{ij}$ from source $i$ to dest $j$ to minimize $\sum c_{ij}x_{ij}$. Always balance first.

Transportation flow: Balance → VAM (init) → MODI (optimal) → reallocate via closed loop if any $d_{ij}<0$ → re-check.

Assignment flow: Pad to square → row min → col min → cover zeros with min lines → if < $n$ lines, adjust uncovered/double-covered → repeat.

Trap: degeneracy in transport (occupied < $m+n-1$) — add $\epsilon$ and continue. Don't get stuck.

W1Row-reduce + classify a system
Solve $\;\begin{cases}x_1 + 2x_2 - x_3 = 4 \\ 2x_1 + 3x_2 + x_3 = 5 \\ 3x_1 + 5x_2 = 9\end{cases}$ — write the solution set.
  1. Augmented: $[A|b]=\begin{bmatrix}1&2&-1&|&4\\2&3&1&|&5\\3&5&0&|&9\end{bmatrix}$.
  2. $R_2\to R_2-2R_1$: $[0,-1,3,|-3]$. $R_3\to R_3-3R_1$: $[0,-1,3,|-3]$.
  3. $R_2=R_3$ identical → row 3 becomes zero after $R_3\to R_3-R_2$. RREF gives one pivot in col 1, one in col 2, col 3 is free.
  4. Let $x_3 = t$. Back-sub: $-x_2 + 3t = -3 \Rightarrow x_2 = 3 + 3t$. Then $x_1 = 4 - 2(3+3t) + t = -2 - 5t$.
  5. Parametric vector form: $\mathbf{x} = \begin{bmatrix}-2\\3\\0\end{bmatrix} + t\begin{bmatrix}-5\\3\\1\end{bmatrix}$.
Answer: infinitely many solutions, parameterized by $t\in\mathbb R$.
Trap: students forget to state "infinite solutions" or write parametric form. Always do both.
W2Linear independence of 3 vectors
Are $v_1=(1,2,3)$, $v_2=(2,5,7)$, $v_3=(1,3,4)$ linearly independent?
  1. Form $A=[v_1\,v_2\,v_3]=\begin{bmatrix}1&2&1\\2&5&3\\3&7&4\end{bmatrix}$. Row-reduce.
  2. $R_2\to R_2-2R_1$: $(0,1,1)$. $R_3\to R_3-3R_1$: $(0,1,1)$. Then $R_3-R_2$: row of zeros.
  3. Only 2 pivots in 3 columns → column 3 is non-pivot → dependent.
  4. Find the relation: from $R_2$ row, $x_2 + x_3 = 0 \Rightarrow x_2 = -x_3$. From $R_1$: $x_1 + 2x_2 + x_3 = 0 \Rightarrow x_1 = -2(-x_3) - x_3 = x_3$.
  5. Take $x_3=1$: $x_1=1$, $x_2=-1$. So $v_1 - v_2 + v_3 = 0$ ✓ (check componentwise).
Answer: linearly dependent; $v_1 - v_2 + v_3 = 0$.
W33×3 Inverse via $[A|I]$
Find $A^{-1}$ for $A=\begin{bmatrix}1&0&2\\2&1&3\\1&0&3\end{bmatrix}$.
  1. Augment: $[A|I_3]$. $R_2\to R_2-2R_1$, $R_3\to R_3-R_1$ → $\begin{bmatrix}1&0&2&|&1&0&0\\0&1&-1&|&-2&1&0\\0&0&1&|&-1&0&1\end{bmatrix}$.
  2. $R_2\to R_2+R_3$: row 2 becomes $(0,1,0\,|\,-3,1,1)$. $R_1\to R_1-2R_3$: row 1 = $(1,0,0\,|\,3,0,-2)$.
  3. Result: $A^{-1} = \begin{bmatrix}3&0&-2\\-3&1&1\\-1&0&1\end{bmatrix}$.
  4. Verify: compute $AA^{-1}$ — must equal $I_3$ (check just diagonal + one off-diag entry to save time).
Answer: $A^{-1}=\begin{bmatrix}3&0&-2\\-3&1&1\\-1&0&1\end{bmatrix}$.
Trap: arithmetic — re-add a row twice and you'll be off by signs. Slow down on row 2.
W43×3 Determinant by cofactor expansion
$\det\begin{bmatrix}2&-3&1\\0&4&-2\\1&-1&5\end{bmatrix}=?$ Expand along column 1 (has a 0).
  1. Cofactor expansion along col 1: $\det = 2\cdot C_{11} + 0\cdot C_{21} + 1\cdot C_{31}$.
  2. $C_{11} = (+1)\det\!\begin{bmatrix}4&-2\\-1&5\end{bmatrix} = 4\cdot 5 - (-2)(-1) = 20 - 2 = 18$.
  3. $C_{31} = (+1)\det\!\begin{bmatrix}-3&1\\4&-2\end{bmatrix} = (-3)(-2) - 1\cdot 4 = 6 - 4 = 2$.
  4. (Sign for $C_{31}$ is $(-1)^{3+1}=+1$.)
  5. $\det = 2(18) + 1(2) = 36 + 2 = 38$.
Answer: $\det = 38$.
Trap: sign on cofactor — always write the checkerboard $(+,-,+)$ pattern before plugging in.
W5Diagonalize 2×2 + compute $A^k$
$A=\begin{bmatrix}7&2\\-4&1\end{bmatrix}$. Diagonalize and write $A^{10}$ as $PD^{10}P^{-1}$.
  1. Char eq: $(7-\lambda)(1-\lambda) - (2)(-4) = \lambda^2 - 8\lambda + 7 + 8 = \lambda^2 - 8\lambda + 15 = 0 \Rightarrow \lambda = 3, 5$.
  2. $\lambda=3$: $(A-3I)=\begin{bmatrix}4&2\\-4&-2\end{bmatrix}\Rightarrow 2x_1+x_2=0\Rightarrow v_1=(1,-2)$.
  3. $\lambda=5$: $(A-5I)=\begin{bmatrix}2&2\\-4&-4\end{bmatrix}\Rightarrow x_1+x_2=0\Rightarrow v_2=(1,-1)$.
  4. $P=\begin{bmatrix}1&1\\-2&-1\end{bmatrix}$, $D=\begin{bmatrix}3&0\\0&5\end{bmatrix}$.
  5. $A^{10}=PD^{10}P^{-1}$ where $D^{10}=\text{diag}(3^{10},5^{10})=\text{diag}(59049,\,9765625)$.
  6. $P^{-1}=\dfrac{1}{(1)(-1)-(1)(-2)}\begin{bmatrix}-1&-1\\2&1\end{bmatrix}=\begin{bmatrix}-1&-1\\2&1\end{bmatrix}$ (since $\det P = 1$).
Sanity: trace $A = 8 = 3+5$ ✓. $\det A = 7 - (-8) = 15 = 3 \cdot 5$ ✓.
W6Markov chain steady state
$P=\begin{bmatrix}0.6&0.3\\0.4&0.7\end{bmatrix}$ (columns sum to 1). Find the long-run steady state.
  1. Check stochastic: col 1 = 0.6+0.4 = 1 ✓, col 2 = 0.3+0.7 = 1 ✓.
  2. Want $Pq=q \Rightarrow (P-I)q=0$. $P-I=\begin{bmatrix}-0.4&0.3\\0.4&-0.3\end{bmatrix}$. (Rows are dependent — expected, since $\lambda=1$.)
  3. From row 1: $-0.4 q_1 + 0.3 q_2 = 0 \Rightarrow q_2 = \tfrac{4}{3} q_1$.
  4. Normalize: $q_1 + q_2 = 1 \Rightarrow q_1(1 + \tfrac{4}{3}) = 1 \Rightarrow q_1 = \tfrac{3}{7}$, $q_2 = \tfrac{4}{7}$.
Answer: $q = (\tfrac{3}{7},\tfrac{4}{7}) \approx (0.429,\,0.571)$.
Trap: if matrix is row-stochastic (rows sum to 1), transpose first or solve $qP=q$ instead.
W7Is $A$ diagonalizable? (Justify)
Is $A=\begin{bmatrix}3&1\\0&3\end{bmatrix}$ diagonalizable?
  1. Triangular → eigenvalues = diagonal: $\lambda=3$ (algebraic multiplicity 2).
  2. $(A-3I)=\begin{bmatrix}0&1\\0&0\end{bmatrix}$. Solve: $x_2 = 0$, $x_1$ free → $v=(1,0)$.
  3. Only 1 independent eigenvector but matrix is $2\times 2$ → can't form $P$ with 2 indep cols.
  4. Geometric multiplicity (1) $<$ algebraic multiplicity (2).
Answer: $A$ is NOT diagonalizable.
W8LP — Graphical max
Maximize $z = 3x_1 + 5x_2$ s.t. $x_1 \le 4$, $2x_2 \le 12$, $3x_1+2x_2 \le 18$, $x\ge 0$.
  1. Boundary lines: $x_1=4$, $x_2=6$, $3x_1+2x_2=18$. Sketch in first quadrant.
  2. Corner points of feasible region: $(0,0)$, $(4,0)$, $(4,3)$ (from $3(4)+2x_2=18$), $(2,6)$ (from $3x_1+2(6)=18$), $(0,6)$.
  3. Evaluate $z=3x_1+5x_2$: $z(0,0)=0$, $z(4,0)=12$, $z(4,3)=27$, $z(2,6)=36$, $z(0,6)=30$.
  4. Max at $(2,6)$ with $z=36$.
Answer: $x_1=2$, $x_2=6$, $z_{\max}=36$.
Trap: forgetting a corner point. Always check the intersection of every pair of binding constraints inside the feasible region.
W9Simplex — one full iteration
Max $z=5x_1+4x_2$ s.t. $6x_1+4x_2 \le 24$, $x_1+2x_2 \le 6$, $x\ge 0$.
  1. Standardize: add slacks $s_1, s_2$: $6x_1+4x_2+s_1=24$, $x_1+2x_2+s_2=6$. Initial BFS: $x_1=x_2=0$, $s_1=24$, $s_2=6$. $z=0$.
  2. Initial tableau — $c_j-z_j$ row: $[5, 4, 0, 0]$. Most positive = 5 → $x_1$ enters.
  3. Ratios: $24/6=4$, $6/1=6$. Smallest = 4 → row 1 ($s_1$) leaves. Pivot at row 1, col $x_1$.
  4. Pivot: $R_1\to R_1/6$ → $(1, 2/3, 1/6, 0, |, 4)$. $R_2\to R_2 - R_1$ → $(0, 4/3, -1/6, 1, |, 2)$.
  5. New $z$ row: $z = 5(4) + 0 = 20$. New $c_j-z_j = [0, 4/3, -5/6, 0]$. Positive entry in $x_2$ col → not optimal yet.
  6. $x_2$ enters. Ratios: $4/(2/3)=6$, $2/(4/3)=3/2$. $s_2$ leaves. Pivot, then check — $c_j-z_j$ all $\le 0$ → optimal.
  7. Final BFS: $x_1=3$, $x_2=3/2$, $z=21$.
Answer: $x_1=3$, $x_2=3/2$, $z_{\max}=21$.
W10Transportation — VAM initial allocation
3 sources × 3 destinations. Costs $C=\begin{bmatrix}8&6&10\\9&12&13\\14&9&16\end{bmatrix}$, supplies $(20,30,25)$, demands $(15,20,40)$.
  1. Balance check: $20+30+25=75$, $15+20+40=75$ ✓.
  2. Row penalties (2nd-min − min): R1: $8-6=2$, R2: $12-9=3$, R3: $14-9=5$. Col penalties: C1: $9-8=1$, C2: $9-6=3$, C3: $13-10=3$.
  3. Max penalty = 5 (R3). In R3, smallest cost = 9 (col 2). Allocate $\min(25, 20)=20$ to $(3,2)$. Col 2 done. Remaining R3 supply = 5.
  4. Recompute penalties without col 2. R1: $10-8=2$, R2: $13-9=4$, R3: $16-14=2$. Cols: C1: $9-8=1$, C3: $13-10=3$.
  5. Max = 4 (R2). Smallest cost in R2 = 9 (col 1). Allocate $\min(30,15)=15$ to $(2,1)$. Col 1 done. R2 remaining = 15.
  6. Only col 3 left. Allocate: R1 → 20, R2 → 15, R3 → 5 (total 40 ✓).
Initial cost: $20\cdot 9 + 15\cdot 9 + 20\cdot 10 + 15\cdot 13 + 5\cdot 16 = 180+135+200+195+80 = 790$. Then run MODI for optimality.
W11MODI optimality check
Using the BFS from W10, run one MODI step: compute $u_i, v_j$, find $d_{ij}$ for empty cells.
  1. Occupied: (1,3), (2,1), (2,3), (3,2), (3,3). Set $u_1=0$.
  2. (1,3): $u_1+v_3 = 10 \Rightarrow v_3 = 10$. (2,3): $u_2+v_3=13 \Rightarrow u_2=3$. (2,1): $u_2+v_1=9 \Rightarrow v_1=6$. (3,3): $u_3+v_3=16 \Rightarrow u_3=6$. (3,2): $u_3+v_2=9 \Rightarrow v_2=3$.
  3. Empty $d_{ij}=c_{ij}-(u_i+v_j)$: (1,1): $8-(0+6)=2$. (1,2): $6-(0+3)=3$. (2,2): $12-(3+3)=6$. (3,1): $14-(6+6)=2$.
  4. All $d_{ij}\ge 0$ → optimal! Stop. Min cost = 790.
Optimal cost: 790.
Trap: sign of $d_{ij}$. If any negative → not optimal, need closed-loop reallocation.
W12Hungarian assignment
Assign 3 workers to 3 jobs to minimize cost: $C=\begin{bmatrix}9&2&7\\6&4&3\\5&8&1\end{bmatrix}$.
  1. Row reduce (subtract row min): R1 min=2 → $(7,0,5)$. R2 min=3 → $(3,1,0)$. R3 min=1 → $(4,7,0)$.
  2. Col reduce: col mins now = (3,0,0) → C1 subtract 3: $(4,0,5)\;(0,1,0)\;(1,7,0)$.
  3. Final reduced: $\begin{bmatrix}4&0&5\\0&1&0\\1&7&0\end{bmatrix}$. Cover all zeros: row 2 has 2 zeros — cover row 2; col 2 covers (1,2); col 3 covers (3,3). That's 3 lines = $n$ → optimal.
  4. Assignment: pick zeros — (1,2), (2,1), (3,3). Each row & col used once.
